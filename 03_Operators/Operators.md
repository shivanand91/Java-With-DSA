# Operators
   ## 1. Arthimatic Operators
     - Binary - (+, -, *, /, %)
     - Unary - (++, --)
        - example: (Increment: a = a + 1, a++, ++a), (Decrement: a = a - 1, a--, --a)
   ## 2. Relational Operators
      - (==), (!=), ((>), (<), (<=), (>=)) 

   ## 3. Logical Operators
       - &&, ||, !

   ## 4. Assignment Operators
       - =, +=, -=, *=, /=
